# claude-ai-chatbot
Created a for Claude AI Document Retrieval chatbot.
Role of Claude AI in Document Retrieval System

In this project, Anthropic’s Claude AI acts as the reasoning engine that generates accurate answers based only on the uploaded documents.

The system uses a Retrieval-Augmented Generation (RAG) architecture, where Claude does not rely only on its pre-trained knowledge but instead uses relevant document context retrieved from a vector database.

This makes the chatbot:

more accurate
more context-aware
reliable for enterprise use cases
safe from hallucinations
